1. alternative_gtf_format_chr_21_and_Y.gtf - A non ENSEMBL GTF file with transcript annotations for human chromosome 21 and Y on GRCh38.
   See this Vignette for an example on how to deal with it: https://rna-pysoforms.readthedocs.io/en/latest/examples/10.dealing_with_different_gtf_files.html


2. counts_matrix_chr21_and_Y.tsv - Counts matrix file with data for transcripts from human chromosome 21 and Y on GRCh38.


3. Homo_sapiens_chr21_and_Y.GRCh38.110.gtf - An ENSEMBL GTF file with transcript annotations for human chromosome 21 and Y on GRCh38.


4. sample_metadata.tsv - An example sample metadata file that can be used to annotate the provided counts matrix.
